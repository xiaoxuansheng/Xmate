## 版本 V2.4.7 — QuickLook 图片标注工具栏 ⭐

### 方案 B：提取共享标注引擎

将截图标注引擎（数据模型 + 工具栏 + Canvas 渲染）提取到 `lib/core/annotate/`。

```
lib/core/annotate/
  annotate_models.dart       ← 标注模型 + 枚举
  annotate_toolbar.dart      ← 工具栏 widget（完整保留）
  annotate_canvas.dart       ← Canvas painter + drawAnnotation
```

兼容性：`screenshot_data.dart` 用 `export` 保持旧 import 路径兼容。

### QuickLook 图片标注

**查看/标注双模式**：右下角编辑按钮 → 进入标注模式（完整 AnnotateToolbar + 标注层 overlay）。

**工具栏布局**：新增 `optionsRowFirst` 参数。QuickLook 传 `true`（选项行在上），截图默认 `false`（原样不变）。

**坐标系统**：标注存储在**图片空间** (0..imgW, 0..imgH)，painter 中完整应用三段变换链：

```
屏幕 → invert(tcValue) → child空间 → translate(crX,crY) → scale(ix,iy) → 图片空间
```

`_updateContainRect()` 每帧显式计算 `BoxFit.contain` 布局，确保窗口 resize / 工具栏弹出 / 全屏切换时标注始终对齐图片。

**交互**：
- 绘图工具：`Listener` (raw pointer) 绕过手势竞技场
- 鼠标模式：tap 选中、pan 移动/缩放/旋转、自动选中新标注
- text/numberTag：单击即放置
- Undo/Redo 快照栈

## V2.4.9 — 编辑模式增强 ⭐

### 裁剪/去背景工具加入工具栏

`AnnotationTool` 枚举新增 `crop` / `bgRemove`，位于橡皮擦右边。截图侧 `hiddenTools: {crop, bgRemove}`。

- **裁剪**：选中工具 → 点击 → 取当前图片边界裁剪 → 窗口自适应尺寸
- **去背景**：选中工具 → 点击像素 → `fillFlood(threshold=12)` 将连续色块替换为透明

### 文本 in-place 编辑

参照截图插件实现：`Positioned` + `Material(transparent)` + `TextField`(无边框、isDense、autofocus)。Enter / 点击外部 → `_ct()` 提交。`_onKeyEvent` 在 `_tx==true` 时放行键盘。

### 工具栏过滤参数

`AnnotateToolbar` 新增 `hiddenTools` (Set) / `hiddenActions` (Set)。QuickLook 传 `{ocr, translate}` + `{'pin'}`。截图不传，不变。

### 非鼠标模式编辑选中标记

`_onPointerDown` 在任何工具模式下优先检测选中标注的 handles 命中，命中后通过 `_onPointerMove` 编辑（move/resize）。

### 画笔/马赛克实时预览

`_AnnotationsPainter` 新增 `freehandPreview` / `mosaicEffect` 参数，paint() 中实时渲染绘制路径。

#### 笔画粗细预设调整

- 旧循环：1 → 3 → 6 → 10 → 1
- 对截图侧和 QuickLook 同时生效

#### 文本样式（`TextStyleKind`）

- 新增枚举：`plain` / `outlineBox` / `filledInverse` / `filledWhite`
- `TextAnnotation` 模型新增 `textStyleKind` 字段（默认 `plain`，保持向后兼容）
- `ToolOptions` 新增 `textStyleKind` 字段 + `_textStyleDropdown()` 下拉菜单

- 新增 `_inverseColor()`：luminance > 0.5 → black，否则 white
- 文本样式切换：改为 4 个图标按钮（Aa / ◫ / ◼ / ◻），位于字体下拉后面

#### QuickLook 编辑模式 Enter 屏蔽

### V2.4.11 — 马赛克/橡皮擦工具修复

#### 问题诊断

马赛克 line 模式三处断裂 + 橡皮擦完全缺失：
- `_onPointerMove` 只给 freehand 积累点
- `_onPointerUp` mosaic 只用 Rect.fromPoints
- Painter 无 saveLayer/BlendMode/order-id，且不传 image

#### Core 微改（截图侧不受影响）

- `_getOrAssignAnnOrder` → `getOrAssignAnnOrder` 公开
- `_drawEraserMask` 从 `_AnnotationPainter` 实例方法提取为 top-level `drawEraserMask`

#### QuickLook 修复

- **`_eraserMasks` 列表**：新增，与 `_ann` 并列
- **`_onPointerDown`**：mosaic-line → 清空+积累点；eraser-line → 同上；eraser rect/ellipse → 走 ds/dc
- **`_onPointerMove`**：扩展 `freehandPts.add` 到 mosaic/eraser line 模式
- **`_onPointerUp`**：mosaic line → `MosaicAnnotation(points:)`；eraser 新增三种模式
- **`_onToolChanged`**：eraser 不清除选择
- **`onClearAll`/`didUpdateWidget`/`_onPin`**：清理 `_eraserMasks`
- **Painter 重写**：新增 `eraserMasks`/`image`/`imageRgba` 参数；saveLayer + order-id 交错排序；eraser line 预览=红色路径；eraser rect/ellipse 预览=红色半透明填充+白边
- **Mosaic 预览**：line 模式不再画 FreehandAnnotation，改为创建 MosaicAnnotation + 传 image 参数

### V2.4.12 — 马赛克虚化度 + 裁剪选框 + 去背景放大镜

#### 马赛克虚化度

- **模型**: `MosaicAnnotation.blurAmount`（默认 1.0）+ `ToolOptions.mosaicBlurAmount`
- **Toolbar**: blur 模式显示 slider（0.3-5.0）+ px 输入框

- **rect/ellipse cellSize**: 修复硬编码 30.0 → `a.cellSize`（截图侧也受益）
