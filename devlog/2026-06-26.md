# 2026-06-26 开发日志

## 版本 V2.2.6 — 滚动截图 dy 估计重构（开发中）

### 修改历程（四轮迭代）

#### 第一轮：移除纵向降采样
- `_toGrayDownsampled` 中 `dh = sh ~/ 2` → `dh = sh`，`sy = dy * 2` → `sy = dy`
- 所有 `_fixedTop`/`_fixedBot` 的 `* 2` 全分辨率映射移除
- dy 不再量化为偶数，精度 2px→1px 步进

#### 第二轮：自适应 maxDy + 方向过滤（C++→Dart wheelDelta 数据通道）
- `screenshot_capture.cpp`：`MouseLLProc` WPARAM = `HIWORD(p->mouseData)` (wheelDelta)
- `flutter_window.cpp`：`WM_XMATE_SCROLL_CAPTURE` → `EncodableMap{"wheelDelta": int32_t}` → method channel
- `annotate_page.dart`：`_onScrollCapture(wheelDelta)` → `addFrame(wheelDelta: wheelDelta)`
- `maxDy = max(wheelDelta × 1.5, roiH/3)`，上限 `roiH × 3/4`
- wheelDelta != 0 时过滤反方向 ZNCC 候选

#### 第三轮：2D 梯度 ZNCC（后被替换）
- 新增 `_gradStrip` (centered difference) + `_znccGrad` (2D ZNCC)
- 删除 `_phaseCorrelate1D`、`_fft1D`、`_znccRefine`
- **结果不理想**——梯度 ZNCC 对文本周期性仍不够辨别力

#### 第四轮：灰度直 SAD + 固定列检测
- **算法替换**：梯度 ZNCC → 灰度 SAD（Sum of Absolute Differences）
- 新增 `_measureFixedCols`：逐列 abs diff，avg<3 标为 fixed（侧边栏等不滚动区域）
- strip 内固定列太多（<25%）→ 跳过该 strip
- SAD 搜索跳过固定列，只累加滚动列的 |a-b|
- 删除 `_gradStrip`、`_znccGrad`、`_variance`

#### UI 改动
- 像素框图标放大 30%（14→18px）
- Debug 面板重构：prev/curr 并排 + 新增区蓝/粉半透明着色 + dy/wheel 大字（3× arial48）
- 滚动截图按钮隐藏（功能开发中）
- 去掉拼接预览图，只显示 debug 面板
