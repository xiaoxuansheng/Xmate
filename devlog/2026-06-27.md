# 2026-06-27 开发日志

## 版本 V2.3.1 — 命令面板优化 + 多项 Bug 修复

### 🐛 Bug 修复

#### 1. 开机启动开关状态刷新
`ToggleAutoStart()` 非提权路径 `ShellExecuteExW(runas)` 立即 return → 查状态时 helper 还没执行完。
修复：`SEE_MASK_NOCLOSEPROCESS` + `WaitForSingleObject(30s)`。
**文件**：`windows/runner/native/tray_icon.cpp`

#### 2. 截图快捷键首次装显示 "Alt"
`_kDefaultKeyId=83` 是 ASCII，`findKeyByKeyId()` 接收 USB HID（S=0x16=22）。
修复：`0x16` + `HotKeyDef.displayString` 用 `findKeyByKeyId`。
**文件**：`screenshot_plugin.dart`, `plugin_base.dart`

#### 3. IME 组字 + 命令面板冲突
- Enter 在 `_handleEarlyKey` 中 `si==-1` 返回 `ignored` 放行给 IME
- `_onSearch` 中 `composing` 检测跳过搜索
- `onSubmitted` → `addPostFrameCallback` 延迟执行
- `textInputAction: TextInputAction.send` 避免框架同步 unfocus
**文件**：`command_palette.dart`

#### 4. Dart tear-off 闭包泄漏
`_handleEarlyKey` 和 `_onSearch` 每次引用创建新闭包 → add/remove 不匹配。
修复：`late final _earlyKeyHandler` 和 `late final _searchListener` 字段缓存。
**文件**：`command_palette.dart`

#### 5. LibreTranslate 安装/启动
Python 安装后 setState 刷新、双检测（isInstalled + checkInstalled）、Scripts 路径修正、状态更新。
**文件**：`translate_settings.dart`, `server_manager.dart`

#### 6. IME 英文 Enter→执行命令后回车失效（核心）
`TextField.onEditingComplete` 默认在 IME completion 窗口内调 `_f.unfocus()` → `clearClient` 损坏 IME 状态。
修复：`onEditingComplete: _c.clearComposing` 替代默认 unfocus，延迟 close 给 IME 时间完成 transition。
**文件**：`command_palette.dart`
