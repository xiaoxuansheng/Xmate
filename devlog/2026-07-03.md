## 第三轮 — 录屏功能增强

### 新功能 1 — 鼠标开关 (showMouse)

键鼠按钮从预留占位变为可用：
- 默认**激活**（亮）→ FFmpeg 不传额外参数 → gdigrab 默认绘制鼠标
- 点击后**取消激活**（暗）→ FFmpeg 传 `-draw_mouse 0` → 录制不含鼠标光标
- 工具栏 + 主设置页均可切换，持久化到 `screenrecording.showMouse`

### 新功能 2 — 音频码率滑块

- 11 档步进：`[48, 64, 80, 96, 112, 128, 160, 192, 224, 256, 320]` kbps
- 主设置页 + 内嵌面板均支持
- 128-192k 标记为 Optimal（绿色）
- 内嵌面板 `_settingsH`: 175→210

### 修复 3 — DropdownButton 红色闪报错

`_dd()` 中 `DropdownButton.value` 不在 `items` 列表时 Flutter 抛 AssertionError。
加 `safeValue = values.contains(current) ? current : values.first` 防护。

### 修复 4 — 录制栏自动隐藏 3s → 1s

## 第一/二轮 — FFmpeg 命令结构 + 奇数尺寸 + overlay

（参见上方 V2.6.8 test list 条目）

### 修复 1 — FFmpeg -i 顺序修正

**根因**：旧代码把 `-c:v` / `-preset` / `-crf` / `-pix_fmt`（视频输出选项）夹在 `-i desktop` 和 `-i audio=...` 两个输入之间。用户实测验证：全部 `-i` 在前、全部输出选项在后的顺序才稳定。

**修复**：
- `recording_state.dart` 拆分 4 个 getter：
  - `ffmpegInputArgs` — gdigrab 输入
  - `audioInputArgs` — dshow 音频输入（可选）
  - `videoOutputArgs` — `-c:v -preset -crf -pix_fmt`
  - `audioOutputArgs` — `-c:a -b:a`（可选）
- `recording_service.dart` 按 `ffmpegInputArgs → audioInputArgs → videoOutputArgs → audioOutputArgs` 组装
- 新增 `_shellCmdString()` / `_shellQuote()` — 生成可复制到 cmd 测试的完整命令行

### 修复 2 — 奇数宽高矫正 (yuv420p)

**根因**：H.264 yuv420p 要求宽高为偶数。用户截图选区经 DPR 缩放后可能出现奇数（如 1023×767），FFmpeg 直接拒绝编码。

**修复**：
```dart
int evenDim(int v) => v < 2 ? 2 : (v & 1) == 1 ? v - 1 : v;
```
- `startWindowRecording()` 中 `physW/physH` → `evenDim()` → `recW/recH`
- overlay 仍使用原始物理尺寸（标注不变），仅 FFmpeg 录制参数使用偶数
- 最小宽度 `v < 2 ? 2` 安全兜底

### 修复 3 — overlay 关闭时机

**旧行为**：`onStatusChanged` 中 `stopped` / `error` → 立即 `_destroyOverlay()`

**新行为**：仅在录制栏 ✕ 关闭时关闭 overlay：
- 录制栏 `onClose` → `await _service.stop()` → 550ms → `windowManager.close()`
- Subprocess dispose → `IpcServer.dispose()` 删除 `sr_status.json`
- Main process `IpcClient._poll()` 检测文件消失 → `onDisconnected` → `_destroyOverlay()`
- 用户停止录制后可继续看到选区 overlay，关闭工具栏时才消失

### FFmpeg 命令结构对比

| | 旧（错误） | 新（正确） |
|------|-----------|-----------|
| **输入** | `-i desktop` | `-i desktop` |
| **输出选项混入** | `-c:v libx264 -preset ultrafast ...` | *(无)* |

| **输出选项** | *(无)* | `-c:v libx264 -preset ultrafast -crf 25 -pix_fmt yuv420p -c:a aac -b:a 128k` |
