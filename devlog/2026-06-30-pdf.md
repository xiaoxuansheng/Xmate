# 2026-06-30 开发日志（续）

## V2.5.2 — QL PDF 预览 + 标注系统 ⭐

### 概述

从 V2.5.1 基础上扩展 QuickLook 支持 PDF 文件预览。使用 pdfrx（PDFium）渲染引擎，集成现有标注系统，支持 per-page 标注、文本选择、跨进程翻译。

### 功能清单

**PDF 预览**：
- pdfrx 引擎渲染（PDFium），支持多页
- 左侧缩略图侧栏（150px 宽），懒加载 10 张每批
- 右侧全页展示，InteractiveViewer 缩放（0.5x~5.0x）
- 鼠标滚轮：Ctrl+滚轮缩放全页，缩略图区滚动翻页
- 页码指示器（右下角 N/M）

**标注系统**（per-page 独立状态）：
- 工具栏始终可见（AnnotateToolbar），9 个工具：鼠标/矩形/箭头/画笔/文字/马赛克/标号/橡皮擦/清除
- 编辑：手柄拖拽（move/resize/rotate）+ Shift 固定比例 + 滚轮调整大小
- 撤销/重做（per-page 栈）
- Delete 删除、Tab 循环选中、WASD 微移（按住加速）
- 隐藏工具：crop、bgRemove、ocr（PDF 不适用）
- 清除全部按钮

**文本选择**（pdfrx 原生）：
- 鼠标工具下，拖拽 PDF 文字释放后自动复制到剪贴板
- 从 PdfPageText（structured text）提取 fragment 坐标，转换 PDF 坐标系到 image 坐标系

**合成/复制/翻译**：
- Ctrl+C 合成当前页 + 标注 → PNG → 原生剪贴板
- 翻译按钮 → `ql_translate_req.json` → `requestTranslate` → 主进程翻译窗口
- 保存按钮（V1 占位） → SnackBar "Multi-page export coming soon"

**快捷键帮助面板**（H 键）

### Per-Page 计数器改造

为使 per-page 标注的 order counter 独立，在 `annotate_canvas.dart` 新增：
- `getOrAssignAnnOrderWithCache(AnnotationShape, Map<String,int> cache, int Function() nextCounter)` — 替代全局 `_sharedOrderCounter`
- `EraserMask.withOrderId(int orderId, ...)` — 接受显式 orderId

## V2.5.3 — PDF 预览迭代：渲染/布局/交互修复

### 概述

V2.5.2 的 PDF 预览存在多个问题：窗口跳变、页面不全、缩放不同步、边界钳制。经多轮迭代修复。

### 渲染与布局

- **360 DPI**：`_renderScale = 360/72 = 5.0`，A4 约 2975×4210 px
- **fit-zoom**：`LayoutBuilder` 计算 `min(viewportW/imgW, viewportH/imgH)` → 始终完整显示
- **居中**：`Transform` 的平移 = `(viewportW - childW) / 2`，缩放后也居中重算
- **窗口尺寸**：`ptW × 5.0 × 0.6 + 150(sidebar)` × `ptH × 5.0 × 0.6 + 88(toolbar)`，clamp to 85% 屏幕
- **PDF 独立定位**：`_centerWindow()` 直接屏幕居中，不用 `QlWindowState` 记忆位置
- **不再调 `_onReady`**：避免 900×650 → 正确尺寸的闪烁

### 缩放与交互重构

- **`InteractiveViewer` → `Transform`**：InteractiveViewer 的边界钳制在缩放后 child < viewport 时把内容拖到左上角。改用纯 `Transform` 自管矩阵
- **手势层提到 Transform 外**：`Positioned.fill` + `HitTestBehavior.translucent` → 全视口收事件，`_toImageSpace` 手动 `Matrix4.inverted(tcValue)` 逆变换
- **显式 `_currentZoom`**：不再依赖 `getMaxScaleOnAxis()`（Z 轴=1 导致缩小失效）
- **中键拖拽平移**：`midLast` delta → `Matrix4.translationValues` 叠加到 `_tc.value`
- **左键仅标注**：空处点击只取消选中，无 pan/文本选区

### 标注溢出

- `_overflow = 200px`：SizedBox 比图片大 400×400，`Canvas.translate(overflow, overflow)` 偏移渲染
- `Clip.none` + `ClipRect` 外层裁剪 → 标注可超出页面，超出部分在视图外可滚动/缩放到达
