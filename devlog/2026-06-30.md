### 关键修复

| 问题 | 根因 | 修复 |

| QL 锁定在已执行文件 | XMatePanel._close() 缺 clear() | app.dart 加 QuickLookPaletteState.clear() |

## V2.5.4 — QL 音频预览 ⭐

### 概述

QuickLook 支持音频文件预览（8 种格式），含完整播放控制、快捷键、Windows Media Foundation 元数据读取。

### 功能清单

**播放控制**：
- 8 种格式：mp3 / wav / flac / ogg / aac / m4a / wma / opus
- 默认不自动播放（`setSource`），Explorer 切文件自动 reload
- 圆形播放/暂停按钮（48×48），加载中显示 hourglass_empty
- 进度条 Slider 拖拽 seek + MM:SS 时间标签

**倍速**：
- 单按钮 + PopupMenu 向上弹出 6 选项
- 0.5× / 0.75× / 1× / 1.25× / 1.5× / 2×，当前项蓝色 ✓ 标记

**音量**：
- 音量按钮 → Overlay 弹出滑块栏（音量图标 + 滑块 + 百分比数字），点击空白消失
- 滑块拖动实时响应（`markNeedsBuild` 修复 Overlay 刷新问题）
- 点击音量图标切换静音/恢复

**循环**：
- `Icons.repeat` 按钮，单击切换 `ReleaseMode.loop` / `ReleaseMode.stop`
- 激活态蓝色边框 + 背景高亮

**信息面板**：
- 点击 ℹ️ → Overlay 弹出 Codec / Sample rate / Channels / Bitrate / Bit depth
- 元数据来源：Media Foundation `IMFSourceReader`（支持 M4A/AAC/OPUS 等全部格式）
- MF 失败 → 回退 Shell property store → 最终 fallback 扩展名推断
- 20 种音频 subformat GUID → 友好名称映射

**快捷键**：

| 键 | 操作 | 长按 |
|----|------|------|
| Space | 播放/暂停 | 否（仅 KeyDownEvent） |
| ← | 快退 1s | 是（KeyRepeatEvent 逐秒连续 seek） |
| → | 快进 1s | 是 |
| ↑ | 增大倍速 | 是（连续递增沿预设档位） |
| ↓ | 减小倍速 | 是 |
| Enter | 打开文件（`cmd /c start`） | 否 |

- `Focus(autofocus: true)` 包裹音频区，`KeyDownEvent` + `KeyRepeatEvent` 双通道处理
- Space/Enter 仅 `KeyDownEvent` 触发（防重复 toggle / 多开）
- Enter 通过 `onOpenFile` 回调到父页面 `_openFile()`

**按钮栏对称布局**：

```
  [🔁]   [1×]   [ ▶ ]   [🔊]   [ℹ️]
  循环    倍速    播放    音量    信息
 (32×32) (32×32) (48×48) (32×32) (32×32)
```

窗口尺寸：**440 × 200**，上方文件名 + 中间进度条 + 下方按钮栏。

**C++ 音频属性解析**：

```
Dart _fetchAudioInfo() → com.xmate/fileops.getAudioProperties(path)
  → C++ GetAudioProperties()
    → MFStartup → MFCreateSourceReaderFromURL(file://…)
      → GetPresentationAttribute(MF_PD_DURATION)     // 时长
      → GetCurrentMediaType(audio stream)
        → GetGUID(MF_MT_SUBTYPE)            → 友好名
        → GetUINT32(…SAMPLES_PER_SECOND)    → 采样率
        → GetUINT32(…NUM_CHANNELS)          → 通道数
        → GetUINT32(…BITS_PER_SAMPLE)       → 位深
        → GetUINT32(…AVG_BYTES_PER_SECOND)  → 码率 (bps)
    ↓ 失败
    → IShellItem2::GetPropertyStore        // 回退 Shell
      → PKEY_Audio_* 属性
```

### 依赖

- `audioplayers: 6.8.1`（含 `audioplayers_windows: 4.4.1`）
- `mfuuid.lib` / `mfplat.lib` / `mfreadwrite.lib`（MF 音频解码链接库）

## V2.5.5 — QL 音频 + 视频预览 ⭐

### 概述

V2.5.5 将 V2.5.4 的音频预览和视频预览合并为最终版本。音频和视频共用相同的 UI 布局（按钮栏对称排列）、快捷键体系（Space/←→/↑↓/Enter）、Overlay 弹窗（音量/倍速/信息）。

### 音频（同 V2.5.4，详见上文）

8 种格式 / 固定 440×200 窗口 / Media Foundation 元数据 / ℹ️ 信息按钮

### 视频

**格式**：16 种（mp4/mkv/webm/avi/mov/wmv/flv/m4v/mpg/mpeg/3gp/ts/vob/ogv）  
**后端**：fvp (mdk-sdk)，registerWith 注册为 video_player 平台实现，D3D11 硬件加速  
**窗口**：动态自适应视频原始分辨率（barH=130px，优先原宽，超屏 98% clamp），不设默认值  
**第一帧 poster**：initialize → play 闪播 50ms → pause，播放前显示视频首帧  

**音频 vs 视频差异**：

| 差异点 | 音频 | 视频 |
|--------|------|------|
| 窗口尺寸 | 固定 440×200 | 动态 = 视频宽 + 130px 控制栏 |
| 第 5 按钮 | ℹ️ 音频信息（MF 元数据） | ℹ️ 视频信息（fvp `getMediaInfo`） |
| Poster | 无 | 闪播 50ms 解码首帧 |
| 尺寸跳变 | 无 | 由 `_onVideoSizeReady` 独家驱动（不预设 480×360） |

**快捷键（音频+视频通用）**：

| 键 | 操作 | 长按 |
|----|------|------|
| Space | 播放/暂停 | 否 |
| ← → | 快退/快进 1s | 是 |
| ↑ ↓ | 增减倍速 | 是 |
| Enter | 打开文件 | 否 |

- `fvp: 0.37.2` + `video_player: 2.11.1`（视频解码）

## V2.5.6 — QL Word 预览（原生 IPreviewHandler 子窗口嵌入） ⭐

### 概述

QuickLook 支持 Word 文档预览（3 种格式），通过 Windows Shell `IPreviewHandler` COM 接口将系统预览 handler 作为原生子窗口嵌入 Flutter 窗口内容区。若系统安装 Office 则显示原生渲染（可滚动/选文字/Ctrl+C），否则回退到 fallback 图标视图。

### 功能清单

**Word 预览**：
- 3 种格式：doc / docx / docm
- 原生子窗口嵌入 `IPreviewHandler`，直接覆盖 Flutter 内容区
- 实例 pool（`unordered_map<int64_t, WordPreviewInstance>`）管理生命周期，无全局裸指针
- 状态机：creating → ready | error（`_doLoad` 已做 registry check，Widget 直接创建）
- 标题栏 Open (Enter) 按钮 → `cmd /c start` 打开 Office 应用 + 关闭 QL 窗口
- rect 同步策略：仅当变化 ≥1 物理像素时才调 `setRect`
- 创建失败 → 回退 `QuickLookFallbackView`

### 架构

```
Dart QuickLookWordView
  → MethodChannel('com.xmate/officepreview')
    create(path, x, y, w, h) → C++ CreateWordPreview
      → AssocQueryString → CLSIDFromString
      → CoCreateInstance(IPreviewHandler)
      → IInitializeWithFile → Initialize
      → CreateWindowEx(WS_CHILD) → SetWindow → SetRect → DoPreview
    setRect(instance, x, y, w, h) → MoveWindow + handler->SetRect
    destroy(instance) → Unload + Release + DestroyWindow
```

### Method Channel 协议

| 方法 | 参数 | 返回 |
|------|------|------|
| `check` | `{ext: ".docx"}` | `bool` |
| `create` | `{path, x, y, w, h}` (物理像素) | `int` (instance handle, 0=失败) |
| `setRect` | `{instance, x, y, w, h}` | `void` |
| `destroy` | `{instance}` | `void` |
