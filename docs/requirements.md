# XMate 项目需求

> 版本：v2.0 | 更新：2026-07-03

## 项目定义

**XMate** = Windows 私人助理工具合集 App，系统托盘常驻，命令面板 + 快捷键驱动。后续扩展到 iOS / Android。

## 当前功能

| 功能 | 状态 | 入口 |
|------|------|------|
| **截图标注** | ✅ 已发布 | Alt+S / 命令面板 "截图" |
| **屏幕录制** | ✅ 已发布 | Alt+R / 命令面板 "录屏" |
| **QuickLook 预览** | ✅ 已发布 | Alt+Q 预览选中文件 |
| **离线翻译** | ✅ 已发布 | 命令面板 "翻译" |
| **文件搜索** | ✅ 已发布 | 命令面板直接输入 |
| **OLE 拖出** | ✅ 已发布 | 文件/文本/图片拖到其他应用 |
| **主题切换** | ✅ V2.7.3 | 设置 → General → 深色/浅色/跟随系统 + 自定义主题色 + 面板透明度 |

## 非功能需求

- 插件化架构，新功能不影响现有功能
- 空闲内存 < 50MB，冷启动 < 2s
- 数据均在本地，不上传
